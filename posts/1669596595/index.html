2. Java常用数据结构以及操作

  1. 数组

    1. 不可变长数组 int[] nums = new int[length];

      操作：

      1. 获取数组的大小：nums.length

    2. 可变长数组 List<Integer> list = new ArrayList<>();

      操作：

      1. 添加: list.add(num);
      2. 获取:list.get(index);
      3. 删除：list.remove(index);
      4. 获取数组的大小:list.size()
      5. 判断数组是否含有元素：list.contains()

  2. 链表

    1. 基本数据结构

    2. 操作：

      1. 获取链表节点的值：int num = node.val;
      2. 获取链表节点的下一个：ListNode temp = node.next

  3. 哈希

    1. HashSet

      • 实现

      • 操作：

        1. 加入元素：hashset.add(num);
        2. 删除元素：hashset.remove(num);
        3. 判断集合是否为空：hashset.isEmpty()
        4. 清空集合：hashset.clear();

    2. HashMap

      • 实现

      • 操作

        1. 加入键值对：map.put(str, num);
        2. 删除键值对：map.remove(str);
        3. 根据键获取值：int numStr = map.get(str);
        4. 根据键获取值，如果键不存在，则返回设定的值：int value = map.getOrDefault(str, default)
        5. 判断集合是否为空：map.isEmpty()
        6. 清空集合：map.clear();

  4. 栈

    • 实现

    • 操作

      1. 入栈：stack.push(num);
      2. 出栈：stack.pop();
      3. 获取栈顶元素:int num = stack.peek();
      4. 判断栈是否为空:stack.isEmpty()
      5. 获取栈的大小:stack.size()

  5. 队列

    • 实现

    • 操作

      • 在队尾端插入元素：queue.offer(num);
      • 移除队首元素：queue.poll();
      • 获取队首元素值：int num = queue.peek();

3.字符串常用操作

  1. 字符串与字符数组之间的转换

    • 字符串转字符数组：char[] ch = str.toCharArray();
    • 字符数组转字符串：String str = new String(ch);

  2. StringBuilder类的操作

    1. 添加字符串：sb.append(str);
    2. StringBuilder与String之间的转换
      1. StringBuilder转换为String：String str = sb.toString();
      2. String转换为StringBuilder：StringBuilder sb = new StringBuilder(str);
